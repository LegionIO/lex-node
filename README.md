# Legion::Extensions::Node


## Installation

This LEX is installed into LegionIO by default. You can disable it with the following setting

```json
{
  "extensions": {
    "node": {
      "enabled": false
    }
  }
}
```

## Usage

This is used by default and you shouldn't have to do anything special to use it
